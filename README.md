# MeeGo-Harmattan-for-Qt6
The goal of the project to run MeeGo Harmattan Components on all desktop and mobile platforms with High-DPI

Tested on Qt 6.6.3

Apps using this project:
[https://play.google.com/store/apps/details?id=org.project.wgs84utm](Latlong & UTM on Google Play Store)
